# Welcome

Signal is ready to use as a backend framework for explicit, deterministic server-side code.

## In One Sentence

Define named queries and mutations, protect them with access rules, and run them on any stateless backend runtime.

- explicit operation names

- clear access rules
- separate layers for HTTP, data, security, and events
